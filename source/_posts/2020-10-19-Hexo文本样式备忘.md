### Note-注意事项

{% endnote %}

{% endnote %}
{% note success %}

{% endnote %}
{% note info %}

{% endnote %}

{% codeblock lang:yaml next/_config.yml %}
# Mermaid tag
note:
  style: flat
  icons: false
  light_bg_offset: 0
{% endcodeblock %}

```

{% endnote %}

{% endnote %}
```

### Mermaid-饼图
{% mermaid pie %}
"Dogs" : 386
"Cats" : 85
"Rats" : 15
{% endmermaid %}
先设置:
{% codeblock lang:yaml next/_config.yml %}
# Mermaid tag
mermaid:
  enable: true
  # Available themes: default | dark | forest | neutral
  theme: forest
{% endcodeblock %}
代码：
```
{% mermaid pie %}
"Dogs" : 386
"Cats" : 85
"Rats" : 15
{% endmermaid %}
```
